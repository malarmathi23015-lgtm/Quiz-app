# Quiz App

A simple terminal-based quiz game built with Python.

## How to Run

Clone the project:
git clone https://github.com/malarmathi23015-lgtm/Quiz-app.git

Go into the folder:
cd Quiz-app

Run the quiz:
python quiz.py

## Features
- 5 multiple choice questions
- Instant correct or wrong feedback
- Final score at the end
